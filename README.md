# VitaLog - Holistic Health Companion

VitaLog เป็นแอปพลิเคชันผู้ช่วยส่วนตัวที่ออกแบบมาเพื่อการดูแลสุขภาพแบบองค์รวม (Holistic Health) ช่วยให้ผู้ใช้งานสามารถติดตามทั้งสุขภาพกายและสุขภาพจิตได้อย่างมีประสิทธิภาพในยุคปัจจุบัน แอปพลิเคชันนี้ทำหน้าที่เป็นเครื่องมือในการบันทึกและวิเคราะห์พฤติกรรมการใช้ชีวิต เพื่อสร้างแรงจูงใจในการปรับเปลี่ยนพฤติกรรมไปสู่การมีสุขภาพที่ดีอย่างยั่งยืน

## คุณสมบัติเด่น (Key Features)

### 1. การติดตามสุขภาพกายและใจ (Health and Mood Tracking)
* บันทึกอารมณ์: เลือกบันทึกความรู้สึกรายวันผ่านระดับอารมณ์ (แย่มาก, ไม่ดี, เฉยๆ, ดี, เยี่ยม) พร้อมระบุกิจกรรมและบันทึกข้อความเพิ่มเติมได้
* บันทึกกิจกรรมรายวัน: ติดตามค่าน้ำหนักตัว (kg), ปริมาณการดื่มน้ำ (ลิตร), ชั่วโมงการนอนหลับ และจำนวนมื้ออาหาร
* การคำนวณ BMI: ระบบคำนวณดัชนีมวลกายภายในตัวแอป เพื่อประเมินภาวะน้ำหนักตัวเบื้องต้น

### 2. ระบบวิเคราะห์และคำแนะนำ (Daily Insights)
ระบบจะทำการวิเคราะห์ข้อมูลที่ผู้ใช้บันทึกในแต่ละวันและแสดงคำแนะนำบนหน้า Dashboard เช่น:
* แนะนำให้ดื่มน้ำเพิ่มหากปริมาณน้ำที่บันทึกต่ำกว่า 1.5 ลิตร
* แนะนำเรื่องการพักผ่อนหากนอนหลับน้อยกว่า 7 ชั่วโมง
* การแจ้งเตือนเรื่องการรับประทานอาหารให้ครบมื้อ

### 3. ระบบปฏิทินและประวัติสุขภาพ (Calendar and History)
* Health Calendar: ระบบปฏิทินที่ช่วยให้ผู้ใช้ย้อนกลับไปดูข้อมูลสุขภาพและอารมณ์ในแต่ละวันได้อย่างละเอียด
* Log History: การสรุปรายการกิจกรรมสุขภาพย้อนหลัง เพื่อให้ผู้ใช้มองเห็นแนวโน้มพฤติกรรมของตนเอง

### 4. ระบบสมาชิกและความปลอดภัย (Authentication)
* รองรับการลงทะเบียน (Register) และการเข้าสู่ระบบ (Login) ผ่าน Firebase Authentication เพื่อความปลอดภัยของข้อมูลส่วนบุคคล

## เทคโนโลยีที่ใช้ (Tech Stack)

* Framework: Flutter (Dart)
* Backend: Firebase Authentication
* Local Storage: Shared Preferences (สำหรับการบันทึกข้อมูลภายในเครื่อง)
* Libraries:
    * table_calendar: สำหรับการแสดงผลปฏิทิน
    * dart:convert: สำหรับการจัดการข้อมูล JSON

## โครงสร้างไฟล์ที่สำคัญ (Project Structure)

lib/
├── screens/
│   ├── login_screen.dart       # หน้าจอเข้าสู่ระบบ
│   ├── register_screen.dart    # หน้าจอสมัครสมาชิก
│   ├── dashboard_screen.dart   # หน้าจอหลักแสดงสรุปผลและคำแนะนำ
│   ├── activity_screen.dart    # หน้าจอบันทึกสุขภาพและดูประวัติ

│   └── auth_service.dart       # ไฟล์จัดการระบบ Authentication
└── main.dart                   # ไฟล์หลักสำหรับรันแอปพลิเคชัน
